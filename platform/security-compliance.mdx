## Core Security Features

Every layer of the Futurum platform enforces security automatically. You do not need to configure or enable these controls — they are active by default on every account.

- **End-to-end encryption** on all data, both in transit and at rest
- **Role-based access controls** for every user type — clinicians, staff, and administrators see only what their role permits
- **Complete audit trails** generated automatically for every patient interaction
- **Encrypted transmission** between patients, doctors, and staff
- **Automatic vital sign sync** to doctor dashboards over encrypted channels

## How Security Is Managed

Futurum handles security at the infrastructure level so your team can focus on care, not configuration.

- Security controls are active from the moment your account is provisioned — no setup required
- Audit logs are generated automatically for every interaction and are available for review from your dashboard at any time
- Access controls are applied based on staff roles and cannot be bypassed at the application level

For more detail on HIPAA-specific protections, see the [HIPAA Compliance](/platform/hipaa) page. For information on the underlying infrastructure, see the [Platform Infrastructure](/platform/infrastructure) page.
